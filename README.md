# teaching-Orange
Orange 是一款功能強大的開放原始碼的免費機器學習軟體，以 Python 編寫，廣泛用於資料科學、機器學習和資料視覺化教學。它適合初學者，暨無需編寫程式碼即可使用其拖放介面，也適合進階使用者結合 Python 程式進行資料處理。

# Orange 3 features
Orange 3 包括以下特性
+ University of Ljubljana, Slovenia, 10 October 1996
+ 視覺化程式設計工具(Visual Programming Tools)**不用寫程式**
+ 執行更加快速(C++)與視覺化操作
+ 提供多種機器學習模組
+ 開放原始碼與跨平台
+ 適用於Python模組
+ 資料預處理
+ 模型訓練
+ 部署
